Avatar

The Avatar component is used to represent a user, and displays the profile picture or initials.

Avatar of different sizes

Avatar is available in 4 different sizes. You can use image in Avatar. You need to include class avatar and for size add class according to size

  • Large: 160px (class: avatar-l)
  • Medium: 100px(class: avatar-m)
  • Small: 48px(class: avatar-s)

Example of Avatar with Text

You can use initial letters of user in Avatar as well. You need to include class avatar-text along with avatar and size class.
